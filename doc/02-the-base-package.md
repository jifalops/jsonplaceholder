# The Base Package

```sh
lib/
  config.dart     # Common values and functions, such as the app's name.
  themes.dart     # The app's theme definitions.
  resources.dart  # The app's resource definitions.
  model/
    post.dart     # Represents a jsonplaceholder post.
    user.dart     # Represents a jsonplaceholder user.
    comment.dart  # Represents a jsonplaceholder comment.
assets/
  fonts/          # Custom fonts.
  images/         # Shared images.
```
